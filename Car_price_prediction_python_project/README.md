# Car Price Prediction Analysis

## Project Overview

This project focuses on analyzing a car dataset to understand the factors that influence car prices. The analysis is performed using Python and various data analysis libraries to explore relationships between different car features and their prices.Through exploratory data analysis (EDA), the project identifies key features that significantly impact car pricing.

## Dataset Information

Important columns in the dataset include:

* **CarName** – Name and brand of the car
* **fueltype** – Type of fuel used (gas or diesel)
* **aspiration** – Engine aspiration type (standard or turbo)
* **doornumber** – Number of doors
* **carbody** – Car body type (sedan, hatchback, etc.)
* **drivewheel** – Drive wheel type (FWD, RWD)
* **enginesize** – Size of the engine
* **horsepower** – Power generated by the engine
* **citympg** – Mileage in city driving
* **highwaympg** – Mileage on highways
* **price** – Price of the car (target variable)

## Project Workflow

### 1. Data Loading

The dataset is imported using **Pandas** and basic inspection is performed to understand its structure.

### 2. Data Cleaning

* Checking for missing values
* Handling data inconsistencies
* Formatting columns if required

### 3. Exploratory Data Analysis (EDA)

Different visualizations are created to analyze relationships between variables, such as:

* Price distribution of cars
* Relationship between engine size and price
* Impact of horsepower on car price
* Fuel type comparison
* Mileage vs price analysis

### 4. Data Visualization

Visualization libraries are used to present insights clearly:

* Histograms
* Scatter plots
* Box plots
  

### 5. Key Insights

The analysis helps identify which factors most influence car prices, such as:

* Engine size
* Horsepower
* Car body type
* Fuel type
* Mileage
