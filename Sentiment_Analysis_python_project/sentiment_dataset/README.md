## Dataset Description

The dataset used in this project contains text reviews along with their corresponding sentiment labels. It is used to train and analyze sentiment classification models.

### Dataset Features

* **Review Text** – The textual content of the user review.
* **Sentiment Label** – Indicates whether the review sentiment is:

  * Positive
  * Negative
  * Neutral (if applicable)

### Dataset Purpose

The dataset helps in understanding how Natural Language Processing (NLP) techniques can be used to analyze opinions and emotions expressed in text data.

### Data Preprocessing

Before performing sentiment analysis, the following preprocessing steps were applied:

* Converting text to lowercase
* Removing punctuation and special characters
* Removing stopwords
* Tokenization and text cleaning

### Use Case

This dataset is used to:

* Perform sentiment classification
* Generate word clouds for positive and negative reviews
* Understand customer opinions from textual data
